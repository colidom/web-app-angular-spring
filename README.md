# web-app-angular-spring

## ANGULAR

### Crear app

```
ng new customers-app
```

### Lanzar servidor

```
ng serve -o
ng serve --open
```

### Generar nuevo componente(manual)

> Desde src/nombre componente ejecutar:

```
ng generate class footer.component
```
